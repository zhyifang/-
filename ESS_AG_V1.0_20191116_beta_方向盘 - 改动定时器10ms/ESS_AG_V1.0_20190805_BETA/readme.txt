1. 目前只支持nrf51822系列；
2. 此软件基于nrfTools， 而nrfTools基于JLINK，所以此软件正常工作需要安装nrfTools并确认JLINK相关驱动已安装；
3. 软件需要合并3个文件，生成all_in_one.hex；如果给工厂烧录，只需要提供工厂all_in_one.hex(文件名不能改），复制到nrf51burner.exe同一目录下，既可以直接点击烧录。
4. 如果不需要bootloader，直接合并softdevice和你的应用2个文件,也是可以烧录的。（现在不需要OTA功能的情况应该比较少！）
